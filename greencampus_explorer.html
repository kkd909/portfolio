Design Decision
At-a-glance compliance without drilling. Before the redesign, school administrators had to open each campus, each job, and each record to determine compliance status. The redesigned dashboard surfaces the compliance signal at the campus level — administrators see what needs attention immediately, and only drill in when action is required.
Same Data — Different Lens
Both views run on the same underlying data model. The facilities admin sees jobs as tasks to execute. The school admin sees the same jobs as compliance obligations to verify. Role-specific views from a single shared system — the same architecture decision made in InternMatch, applied here to regulatory workflows.
